# myTrack – Übersicht

myTrack verwaltet IMEI-bezogene Ressourcen über eine REST-API.

- Stack: Spring Boot, Spring MVC, Spring Security, Spring Data JPA, Jakarta Validation, MapStruct, Lombok, Java 23
- API-Basis: /api/v1
- AuthZ: Rollen (z. B. GOD) + Ownership Checks
- Inhalte:
  - API.md – vollständige REST-API
  - SECURITY.md – Sicherheit, AuthN/AuthZ, CORS
  - SETUP.md – Lokales Setup & Build
  - CHANGELOG.md – Änderungen
  - LICENSE.md – Lizenz
